## VideoJS Theme Kit
An interactive plugin


## Installation  
`npm install videojs-theme-kit`

## Usage 
```
import 'videojs-theme-kit';

player.on('ready',()=>{
  player.theme({skin: SKIN_NAME})
})
```

## Options

Available SKIN_NAME names are provided as :-

1. 'slate'.
2. 'spaced'.
3. 'sleel'.

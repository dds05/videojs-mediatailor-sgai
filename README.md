## VideoJS Theme Kit
An interactive plugin to customise UI of the player


## Installation  
`npm install videojs-theme-kit`

## Usage 
```
import 'videojs-theme-kit';

player.on('ready',()=>{
  player.theme({skin: SKIN_NAME})
})
```

## Options

Available SKIN_NAME names :-

1. 'slate'.
2. 'spaced'.
3. 'sleel'.
4. 'zen'
